background color - color de fondo

si no se especifica un color de fondo, el fondo es trasparente.

css trata cada elemento html como un cuadro y la propiedad background-color establece el color deo fondo para el cuadro . se deve especificar un color de fondo de las mismas tres formas en que puede especificar colores del primer plano: valores rgb, codigo exadecimal y nombre de colores, tambien hemos utilizado la propirdad padding para separar el texto de los bordes de los cuadros. esto hace que sea mas facil de leer.
