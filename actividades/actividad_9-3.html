el elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel de bloque. por ejemplo, pude crear un elemento <div> para que contenga todos los elementos del encabezado del sitio (el logotipo y la navegacion), o puede crear un element <div> para que contenga informacion en l bloque.

en un navegador, el contenido del elemento <div> comenzara en una nueva linea, pero aparte de esto no hara ninguna diferencia en la presentacion de la paguina. sin embargo, el uso de un atributo id o class en el elemento <div> sognifica que ouede crear reglas de escritorio css para indicar cuanto esoacio deve ocupar el elemento <div> en la pantalla y cambiar la apariencia de todos los elementos contenidos en el. tambien puede facilitar el seguimiento del codigo si ha utilizado elementos <div> para contener cada seccion de la paguina.

dado que pude haber varios otros elementso dentro de un elemento <div>, puede ser util agrgar un comentario despues de la etiqueta </div> de sierre. esto le oermite ver claramente a que etiqueta de apertura se supone que corresponda.
