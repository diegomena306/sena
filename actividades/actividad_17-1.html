conceptos clave en el posicionamiento de elementos

construccion de bloques

css tratar cada elemento html como si estuviera en su propio cuadro. este cuadro sera un block-level o un inline box. los cuadros block-level comienza en una nueva linea y actuan con los principales bloques de construccion de cualquier diseño, mientras que los inline-box fluyen entre el texto circundante. puede controlar la cantidad de espacio que ocupa cada caja estableciendo el ancho e los cuadros (y a veces la altura tambien). para separar los cuadros, pueden usar bordes, margenes, relleno y colores de fondo.

normal plow

cada elemento de block-level se encuentra en la parte superior de la siguiente. puesto que esta es la forma predeterminada en que los exploradores tratan los elementos html, no se necesita una propedad css para indicar que los elementos deben aparecer en el flujo normal.
