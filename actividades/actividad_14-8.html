letter and word apacing

by tps sena

el kerning es el termino que utiliza los tipografos para el espacio entre cada letra. puede controlar el espacio entre cada letra con la propiedad: letter-spacing. es particular mente para aumentar el kerning cuamdo se encabesa o frace esta en mayuscula. si em texto esta en un caso nornal, aumentar o disminuir el kerning pude dificultar la lectura.

el intervalo predeterminado

cuado se especifica un valor para estas propiedades, se debe dar em ems. el intervalo predeterminado entre las palabras se establece por el tipo de letra (a menudo alrededor de 0,25 em), y es poco probable que deba cambiar esta propiedad con regularidad.
