un video e un cachorro jugando en la nieve

la etiqueta video: para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source> dentro del elemento <video> (esto debe reemplazar el atributo src en la etiqueta ≷video> de apertura.) tambien puede utilizar varios elementos <source> para especificar que el video esta disponible en diferentes formatoa. (debido a un error en el ipad, debe proporcionar el video mp4 como el primer formato. de lo contrario, puede que no inicie.)

src: este atributo especifica la ruta de acceso al video.

type se debe utilizar este atributo para decirle al navegador que formato es el video. de lo contrario, descarga parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda).

codecs: el codec que se uso para codificar el video se suministra dentro del atributo type. tenga en cuenta el uso de comillas simple, asi como comillas dobles en el atributo de tpo, cuando se suministra
